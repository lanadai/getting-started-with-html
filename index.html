How the Web Works

Major components of the web:

1. Web Browser
2. Internet
3. Servers

User's performs a search via the browser and HTTP requests are made to servers. The server finds the appropriate HTML document and sends it back to the user's computer where the browser interprets the page and displays it on the user's screen.

HTML

HTML stands for Hypertext Markup Language.

-text content
-markup describes what you see.
-references to other docs, videos, images
-links to other pages like This video

HTML Tags and Elements

There's an opening and closing tag i.e. <opening tag> content </closing tag> The opening + closing tag + contents is an Element.

Inline vs Block Elements

-Inline elements occupies the space bounded by the tags that define the element.
-Block elements normally start and end with a new line, when displayed in a browser.
